# Mohanbabu S Portfolio

A clean, responsive personal portfolio landing page built with a single static `index.html` file.

## Overview

This project is a simple portfolio website for **Mohanbabu S**, a BE CSE (AI & ML) student and aspiring AI engineer. It includes:

- Hero section with introduction and call-to-action
- About section with personal summary and education stats
- Skills section with skill highlights
- Projects section showcasing an AI project
- Contact section with email link
- Smooth animations and responsive layout

## Features

- Fully static HTML/CSS page
- Mobile-friendly responsive design
- Minimal, modern aesthetic
- Intersection Observer reveal animation
- Built using only HTML and inline CSS

## Usage

1. Open `index.html` in your browser.
2. Or host the file on any static site service (GitHub Pages, Netlify, Vercel, etc.).
